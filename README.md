# E-Commerce Sales Insights Analysis

## Project Overview

This project performs structured data analysis on an e-commerce transactional dataset.  
The dataset is derived from a relational SQL database (Project 3) and exported as CSV files to simulate a real-world production data workflow.

The objective is to transform raw transactional data into meaningful business insights using Python.

---

## Objectives

- Load structured CSV datasets using Pandas
- Perform data cleaning and missing value handling
- Merge relational tables for analysis
- Calculate revenue metrics
- Perform customer and product-level aggregation
- Visualize sales performance
- Extract business-level insights

---

## Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Dataset Structure

The project includes the following relational tables:

- Customers
- Products
- Orders
- Order_Items
- Payments

These tables simulate a real-world e-commerce database export.

---

## Key Analysis Performed

- Missing value handling
- Multi-table merging
- Revenue calculation (Quantity × Price)
- Customer revenue analysis
- Product performance analysis
- Payment method distribution analysis
- Business insight extraction

---

## Business Insights

- Identified high-value customers
- Determined top-performing products
- Analyzed revenue distribution
- Evaluated preferred payment methods

---

## Conclusion

This project demonstrates an end-to-end data analysis workflow including data cleaning, transformation, aggregation, and visualization — suitable for real-world business decision-making scenarios.
